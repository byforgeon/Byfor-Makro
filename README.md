# Byfor-Makro
Byforgeon ekibi tarafından geliştirilmiş oto tıklama uygulamasıdır.
Klasörde makro diye bir sanal ortam vardır makro.py daki kütüphaneler sistem deki diğer kütüphanaler ile çakışmasın diye sanal ortam kullandık.
Bu makro kodumda pynput kütüphanesini kullanarak makro yaptım ve geliştirdik.
Kodumuza baka bilirsiniz.
